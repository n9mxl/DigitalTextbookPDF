# 디지털 교과서 PDF 변환기 (안전 + 다운로드 불필요)

디지털 교과서 캡쳐 이미지를 **책 페이지 크기 기준**으로 자동 자르고,
**캡쳐 순서 그대로 PDF**로 변환해주는 Streamlit 앱입니다.

## 특징
- [수정] 안전 버전: DOM 오류 최소화, 업로드/PDF 생성 단계 분리
- [수정] 추가 다운로드 불필요: requirements.txt 최소화
- [수정] CPU/태블릿 최적화: Pillow 안정 버전 사용
- 캡쳐 순서 그대로 PDF 페이지 유지
- 고화질 유지 (RGB + LANCZOS + PDF quality=100)

## 사용 방법
1. ZIP 파일 업로드 (캡쳐 순서대로)
2. 책 페이지 픽셀 수 입력 (가로 × 세로)
3. PDF 생성 버튼 클릭 → PDF 다운로드
